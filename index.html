Perjalanan

Perkenalkan nama lengkap saya Kadek Ari Putra Yasa, kalian bisa memanggil saya ARI. Saya berasal dari desa Bengkel, kecamatan Busungbiu, kabupaten Buleleng, Dan saya dari SMK Pariwisata Banyuatis.

Pertama-tama, sebelum saya masuk ke Universitas Pendidikan Ganesha(UNDIKSHA), saya sebenarnya tidak ingin kuliah. Saat saya duduk di bangku SMK saya berpikir kalau sudah masuk SMK sesudah saya lulus, saya harus bekerja. Tetapi kenyataannya tidak, saya bisa untuk masuk kuliah lagi untuk mengasah skill saya. Dan di saat saya SMK saya mengalami miss pengambilan jurusan(jurusan yang saya ambil adalah MULTIMEDIA). Di mana saat itu saya berfikir kalau MULTIMEDIA itu dapat pelajaran Program. Tetapi kenyataannya tidak, saya hanya mendapat pelajaran Program di semester 1 dan 2 saja. Dan dari sanalah saya berfikir untuk tidak langsung bekerja, tetapi melanjutkan kuliah.

Sebelum saya masuk ke Universitas Pendidikan Ganesha, Fakultas Teknik dan Kejuruan, Jurusan Teknik Informatika, dan Prodi Ilmu Komputer, saya cukup banyak mengalami tantangan seperti, gagal dari jalur SNBP dan SNBT. Untungnya saya bisa diterima di jalur mandiri Universitas Pendidikan Ganesha.
Alasan saya berkuliah di Universitas Pendidikan Ganesha adalah karena UNDIKSHA adalah salah satu PTN terbaik di Bali, dan UNDIKSHA dekat dengan dengan daerah saya.

Harapan dan Tanggapan
